# Supply-Chain-Optimization
- A repository for applying ML to optimize supply chain management, covering demand forecasting, inventory, logistics, and supplier selection. Includes Jupyter notebooks, Python scripts, and datasets for real-world applications.

A practical machine learning project focused on optimizing key supply chain operations such as demand forecasting and inventory management. The project demonstrates how data-driven techniques can reduce costs and improve decision-making.

---

## 🔍 Project Overview

This project applies data analysis, machine learning, and optimization techniques to solve real-world supply chain problems.

Key focus areas:
- Demand Forecasting  
- Inventory Optimization  
- Cost Minimization  

---

## 🚀 Features

### 📊 Demand Forecasting
- Used historical sales data to predict future demand  
- Implemented regression-based models using Scikit-learn  
- Improved planning accuracy  

### 📦 Inventory Optimization
- Applied Linear Programming using PuLP  
- Optimized stock levels while satisfying demand constraints  

### 📉 Cost Optimization
- Built optimization model to minimize inventory-related costs
